# Responsible AI in Healthcare

This project explores responsible artificial intelligence techniques in healthcare using machine learning models built on patient readmission data.

## Project Areas

### Fairness and Bias Mitigation
- Evaluated model fairness across demographic groups
- Measured disparities using Fairlearn
- Applied bias mitigation techniques
- Assessed allocation harms in healthcare predictions

### Explainable AI (XAI)
- Used SHAP for global model interpretability
- Used LIME for local prediction explanations
- Improved transparency and trustworthiness of model decisions

## Technologies
- Python
- Scikit-Learn
- Fairlearn
- SHAP
- LIME
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Outcomes
- Identified demographic performance disparities
- Improved model transparency
- Demonstrated responsible AI practices in healthcare analytics
